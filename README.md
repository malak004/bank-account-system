# 🏦 Python Bank Management System

A simple bank account management system built using **Python Basics**.

## Features

- Register a new account
- Login
- Check balance
- Deposit money
- Withdraw money
- Transfer money
- Change password
- Logout

## Concepts Used

- Functions
- Dictionaries
- Loops
- Conditional Statements
- Input Validation

## How to Run

```bash
python bank.py
```

## Notes

This project was built using Python basics only.

It does **not** use:
- Object-Oriented Programming (OOP)
- Databases
- File Handling
- External Libraries
